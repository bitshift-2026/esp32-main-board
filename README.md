# SMART COMPASS

Smart Compass is a GPS-enhanced compass that lets you save locations with one tap. Its analog dial then guides you back, always pointing directly toward your chosen destination—simple, intuitive navigation without maps.


## COMPONENTS

- ESP32
- Magnetometer
- GPS antena
- LCD Display
- Jumper wires
- Breadboard
